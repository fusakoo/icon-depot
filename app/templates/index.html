Welcome to Icon Depot

Directories

  • / : this page
  • [GET]
    • /icons-available : get a list of icons that can be queried
    • /categories-available : get a list of categories that can be queried
    • /tags-available : get a list of tags that can be queried
    • /icon/<icon_name> : specify the name of icon and get a dict/json for the icon's html
  • [POST]
    • /icons + JSON request : similar to /icon, but get html for multiple icons
